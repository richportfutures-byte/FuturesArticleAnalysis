# Contract Prompt Library

This bundle contains five contract-specific prompt libraries and one master deployment guide.

Included contracts:
- NQ
- ZN
- GC
- 6E
- CL

Each contract folder contains the full set of prompt blocks needed to:
1. search and screen for relevant articles
2. cluster competing media narratives
3. run deep article analysis
4. translate the article into contract-specific implications
5. apply deployment guidance so analysis informs but does not dictate trades

Master guide included:
- `docs/source_of_truth/master_guide/Master_Deployment_Guide_By_Contract_v2.docx`

Operator rule:
Use the workflows to shape relevance, bias, scenario framing, and confirmation logic. Do not use them as autonomous trade directives.
