# Deep Analysis Protocol

This is the full audit layer.

## Required steps
1. Extract the article's core claim.
2. Separate facts, inference, speculation, and rhetoric.
3. Identify what is genuinely new versus recycled context.
4. Map first-order and second-order effects.
5. Explain the transmission mechanism into 10-Year Treasury Note Futures (ZN).
6. Steelman the strongest opposing interpretation.
7. Decide whether the article changes price expectations, path expectations, or merely narrative framing.
8. Assign a horizon: headline reaction, same-session, multi-session, or context only.
9. State what evidence would confirm or invalidate the read.
10. End with an explicit statement about how the output should inform but not dictate trading.

## Failure modes to avoid
- overstating certainty
- collapsing several drivers into one vague story
- confusing narrative plausibility with market importance
- ignoring whether the market may already have priced the information
- using article logic as a substitute for actual market confirmation
