# Pre-Trade SOP

1. Identify the article or narrative cluster.
2. Run the deep-analysis prompt.
3. Record the driver hierarchy.
4. Record the horizon and pricing assessment.
5. Write down the minimum confirmation required.
6. Check current market state: trend, balance, squeeze, exhaustion, or event lock.
7. Decide whether the workflow is providing actionable context, weak context, or no usable edge.
8. Only then evaluate a setup in the contract.
9. If price rejects the thesis, stand down or reduce conviction.
